Data Upload Guide

CSV File Requirements

File Format:

  • CSV format (.csv extension)
  • Header row with column names
  • One record per day
  • No missing critical data points

Column Names (flexible):

  • Date: date, Date, DATE
  • Temperature: temp, temperature, air_temp
  • Solar Radiation: solar, rad, radiation
  • Humidity: rh, humidity, relative_humidity
  • Wind: wind, wind_speed, ws

Sample CSV Structure

Date,Temperature,Solar_Radiation,Humidity,Wind_Speed
2024-01-01,22.5,18.2,65,2.1
2024-01-02,24.1,19.8,60,1.8
2024-01-03,21.3,16.5,70,2.4
...

Understanding Results

Statistics Explained

  • Average ET₀: Mean daily evapotranspiration
  • Maximum ET₀: Highest daily value
  • Minimum ET₀: Lowest daily value
  • Standard Deviation: Measure of variability

Chart Interpretation

  • Daily Values: Raw ET calculations
  • Smooth Line: 5-day rolling average
  • Seasonal Patterns: Look for trends
  • Peak Values: Summer maximums

Troubleshooting

File Upload Issues

  • Ensure file is in CSV format
  • Check that column headers match expected names
  • Verify no special characters in data
  • File size should be under 10MB

Calculation Errors

  • Check for missing required columns
  • Ensure numeric data is properly formatted
  • Remove any text in numeric columns
  • Verify date format (YYYY-MM-DD preferred)

Result Issues

  • Very high/low values may indicate data errors
  • Check units: temperature in °C, radiation in MJ/m²/day
  • Missing data points will be skipped
  • Compare results with local ET estimates

Tips for Best Results

Data Quality

  • Use high-quality weather station data
  • Ensure consistent measurement intervals
  • Verify data for obvious errors

Time Period

  • Use complete growing seasons
  • Include multiple years for patterns
  • Consider seasonal variations

Location

  • Use data from nearby weather stations
  • Consider elevation differences
  • Account for local microclimates
