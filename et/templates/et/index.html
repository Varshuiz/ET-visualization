Priestley-Taylor ET Calculator

Calculate evapotranspiration using the Priestley-Taylor method with temperature and solar radiation data

How to Use

  • Upload CSV files from ACIS (Applied Climate Information System) or similar weather data sources
  • Ensure your CSV contains columns for Air Temperature (°C) and Solar Radiation (MJ/m²/day)
  • The tool will calculate ET₀ using the Priestley-Taylor method
  • View results in table format and download processed data
  • For more comprehensive analysis, try the Comparison Calculator
