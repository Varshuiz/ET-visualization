Use this as planning guidance. Adjust by crop stage, soil type, and local field conditions.

Basis: daily reference evapotranspiration (ET₀) uses the FAO-56 Penman–Monteith equation. Global shortwave (Rₛ) uses Open-Meteo forecast daily shortwave_radiation_sum (MJ/m²) at the city coordinates when the API returns a value; only if that is missing does the model fall back to a simple temperature-based Rₛ estimate. Relative humidity prefers the MSC citypage value when present; otherwise it uses Open-Meteo relative_humidity_2m_mean for that date (not a fixed 65%). u₂ prefers MSC wind (speed/gust, km/h) converted consistently with Open-Meteo wind_speed_10m_max, then Open-Meteo forecast wind if needed; a 2.0 m/s placeholder applies only if both fail.

Crop type picks a crop-specific curve of growing-degree-days (GDD, base 5 °C). Each day, GDD from forecast temperatures is added to a running total; that total falls into a growth stage, and a stage factor (roughly 0.55–1.25 depending on crop and stage) multiplies ET₀ for that day. That approximates higher crop water use in peak growth and lower in early or late season—it is a simple stage scaling, not a full crop-growth or soil-water model.

Soil type applies a soil factor (loam = 1.00) to the cumulative irrigation need and to the teal chart line: sandier profiles use factors above 1 (more suggested irrigation, looser/shallower storage in this shorthand), while clay-rich types use factors below 1 (less suggested, more water retained in this shorthand). Precipitation and ET₀ are unchanged; only the interpreted irrigation total is scaled.

The irrigation suggestion sums, day by day, only the shortfall when stage-adjusted ET exceeds forecast precipitation (no credit for “extra” rain beyond the daily need in this simple ledger).
